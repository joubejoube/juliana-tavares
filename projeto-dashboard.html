Dashboard de Analytics Educacional

Uma plataforma full-stack construída do zero pra transformar dados espalhados em decisões estratégicas, acompanhando mais de 700 estudantes em tempo real.

Meu papelConcepção, desenvolvimento e manutenção (ponta a ponta)
ContextoIsmart (Fundação Telles), 2026
StackJavaScript, API Anthropic (Claude), Vercel, GitHub, Monday.com, Google Sheets

O problema

Um programa educacional com mais de 700 estudantes distribuídos em 4 cidades gera muitos dados: presença, entregas, avaliações, satisfação, observações de campo. Só que esses dados viviam espalhados em planilhas, quadros de gestão e formulários, sem visão consolidada.

Na prática, isso significava que cada decisão estratégica dependia de horas de consolidação manual, e a diretoria enxergava o programa com semanas de atraso.

A solução

Construí uma plataforma própria de analytics que conecta todas as fontes de dados do programa e apresenta os indicadores em tempo real, com painéis pensados pra cada público: visão estratégica pra diretoria e visão operacional pros times regionais.

  • Integração de múltiplas fontes: Monday.com e Google Sheets consolidados via proxy serverless, sem depender de exportação manual;
  • IA embarcada: integração com a API da Anthropic (Claude) pra análises qualitativas a partir de respostas abertas;
  • Deploy contínuo: hospedagem no GitHub com publicação automática na Vercel, permitindo evolução constante da ferramenta;
  • Monitoramento de satisfação: acompanhamento por ciclo da percepção dos estudantes, com taxa de resposta e recortes por cidade.

Como construí

Comecei pelo diagnóstico com quem usava os dados no dia a dia, mapeando quais perguntas precisavam de resposta rápida. A primeira versão nasceu como protótipo num hackathon interno e evoluiu de forma iterativa: cada ciclo de uso gerava ajustes de indicadores, novos painéis e melhorias de experiência.

Todo o desenvolvimento foi feito com IA como ferramenta de trabalho, do código à documentação, o que me permitiu entregar uma plataforma completa sem time de tecnologia dedicado.
